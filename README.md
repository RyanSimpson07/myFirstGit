Title: My First Git
Description: This is my first Git and Node.js project created for my full stack development course
Install Instructions:
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the project folder:
   ```bash
   cd myFirstGit
   ```
3. Install the project dependencies:
   ```bash
   npm install
   ```

4. Run Instructions
Run the following command in the project directory:
```bash
node index.js
```
